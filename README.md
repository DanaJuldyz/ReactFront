## How To Use

1. Fork this repository and clone it to your local machine.

2. Install dependencies

```bash
$ yarn
```

3. Edit `portfolio.js`

4. Start your development server

```bash
$ yarn dev
```

## Linking Portfolio to Github

```javascript
  //  portfolio.js
  githubUserName: 'YOUR GITHUB USERNAME HERE',
```
